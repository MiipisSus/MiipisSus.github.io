米花機 Miifachi

米花機是一個多功能的 Discord 機器人，從實用向的投票、抽獎，到娛樂向的抽籤、星座占卜，以及播放音樂的功能都有！
另外也有利用 API 串接訓練好的米花機 AI，可以在 Discord 上與米花機互動。

更詳細的資訊請見 GitBook↓

DCAI

DCAI 是一款針對 Character ai 的多項聊天功能，整理並串接到 Discord 機器人上的背景運行程式。
該程式可以一次運行多個 DCAI Bot ，並與它們在 Discord 上進行對話。

更詳細的資訊請見 Github ↓
